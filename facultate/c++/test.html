P2.1
Convertirea in binar, hexa si in baza 10.

Exemplul nostru: Numărul 13 (este deja in baza 10)

1. ÎN BAZA 2 (BINAR) - METODA "ÎMPĂRȚIREA LA 2"

Împărțim numărul la 2 și ținem minte restul (care va fi mereu 0 sau 1).

  • 13 / 2 = 6 (rest 1)
  • 6 / 2 = 3 (rest 0)
  • 3 / 2 = 1 (rest 1)
  • 1 / 2 = 0 (rest 1)
Acum, citim resturile de jos în sus.
Rezultatul: 1101
2. ÎN BAZA 16 (HEXAZECIMAL)

Aici avem cifre de la 0 la 9, iar apoi litere: A=10, B=11, C=12, D=13, E=14, F=15. Metoda este împărțirea la 16.

  • 13 / 16 = 0 (rest 13)
Ne uităm în listă: 13 corespunde literei D.
Deci, 13 în hexa este D.

P2.3
Care este sintaxa generala de declarare a unei functii / variabile sau constante.
( in continuare are chef de glume )
= Variabile Gândește-te la o variabilă ca la o cutie pe care pui o etichetă (numele) și în care ții ceva (valoarea). Trebuie să îi spui calculatorului ce formă are cutia (tipul de date). Sintaxa: tip_date nume_variabila = valoare; Exemple: int scor = 100; (pentru numere întregi) float pret = 9.99; (pentru numere cu virgulă) char litera = 'A'; (pentru un singur caracter) = Constante Sunt variabile "bătute în cuie". Odată ce le-ai dat o valoare, nu o mai poți schimba în restul programului. Se folosește cuvântul cheie const. Sintaxa: const tip_date NUME_CONSTANTA = valoare; Exemplu: const float PI = 3.14; = Funcții O funcție este ca o mașinărie mică: îi dai materiale (parametri), ea lucrează și îți scoate un produs finit (rezultatul returnat). EXEMPLU:
exemplu.c
int adunare(int a, int b) {
    return a + b;
}
